# 智能投资助手用户指南

## 系统概述

智能投资助手是一个为投资者提供数据分析、估值模型和投资组合管理的综合平台。本指南将帮助您快速上手并充分利用系统的各项功能。

## 快速开始

### 启动系统

1. **启动后端服务**
   ```bash
   cd backend
   npm start
   ```

2. **启动前端应用**
   ```bash
   npm run dev
   ```

3. 在浏览器中访问前端应用（默认地址：http://localhost:5173）

## 核心功能使用指南

### 1. 股票估值分析

#### 查看股票估值
- 导航至「估值分析」页面
- 在搜索框中输入股票代码（例如：600519）
- 点击「分析」按钮查看详细估值结果

#### 估值指标解读
- **PE (市盈率)**: 股价与每股收益的比率，反映市场对公司盈利能力的预期
- **PB (市净率)**: 股价与每股净资产的比率，反映公司资产的溢价程度
- **PS (市销率)**: 股价与每股销售额的比率，适合评估高增长但尚未盈利的公司
- **内在价值**: 通过现金流折现模型计算的理论价值
- **安全边际**: 市场价格与内在价值之间的差距，作为投资决策的安全缓冲

#### 行业对比
- 在估值分析页面，点击「行业对比」标签
- 查看当前股票与同行业其他公司的估值指标对比
- 分析股票在行业中的相对估值水平

### 2. 投资组合管理

#### 创建投资组合
- 导航至「投资组合」页面
- 点击「新建组合」按钮
- 输入组合名称和描述
- 点击「创建」完成组合创建

#### 添加持仓
- 在投资组合详情页面
- 点击「添加持仓」按钮
- 输入股票代码、数量和购买价格
- 点击「确认」添加持仓

#### 查看组合绩效
- 在投资组合列表中，选择要查看的组合
- 点击「绩效分析」标签
- 查看组合的总体收益率、年化收益率、波动率等指标
- 分析组合的历史表现趋势

#### 资产配置分析
- 在投资组合详情页面，点击「资产配置」标签
- 查看各股票在组合中的权重分布
- 分析组合的行业分布情况
- 根据分析结果优化资产配置

### 3. 投资决策支持

#### 获取投资建议
- 在股票详情页面，查看「投资建议」部分
- 根据系统的估值模型和市场分析，获取买入/持有/卖出建议
- 查看建议的详细理由和支持数据

#### 风险管理建议
- 查看组合的风险指标（如最大回撤、夏普比率等）
- 根据系统提供的风险管理建议，调整组合以降低风险
- 设置止损位和利润目标，制定完善的风险控制策略

## 常见问题解答

### Q: 如何更新持仓信息？
A: 在投资组合详情页面，找到要更新的持仓，点击「编辑」按钮，修改数量或价格后保存。

### Q: 系统的估值结果可靠吗？
A: 系统的估值模型基于公开数据和多种估值方法计算得出，但仅供参考，不构成投资建议。投资决策请结合自身风险承受能力和市场判断。

### Q: 如何删除不需要的投资组合？
A: 在投资组合列表页面，找到要删除的组合，点击「删除」按钮，确认后即可删除。

### Q: 系统支持实时行情数据吗？
A: 当前版本使用的是模拟数据。在实际应用中，系统可以接入第三方金融数据API获取实时行情。
